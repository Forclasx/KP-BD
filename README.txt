README - СУХОФРУКТЫ НА ВЫСШЕМ УРОВНЕ
=======================================

Этот файл содержит логины, пароли и пошаговую инструкцию по настройке сайта на хостинге.

----------------------------------------
1. Логины и пароли
----------------------------------------

1.1. База данных (MySQL):
- Хост: localhost (или адрес, предоставленный хостингом)
- Имя базы данных: dryfruits_shop_db
- Пользователь: root
- Пароль: (пустой на XAMPP, для хостинга создайте нового пользователя с паролем)

1.2. Административная панель (admin.php):
- Логин: admin
- Пароль: admin123

1.3. Тестовый пользователь (для входа через login_user.php):
- Логин: testuser
- Пароль: test123



**Важно**: На хостинге рекомендуется сменить все пароли на более надёжные, чтобы избежать взлома!

----------------------------------------
2. Инструкция по настройке сайта на хостинге
----------------------------------------

Следуйте этим шагам, чтобы развернуть сайт "Сухофрукты на высшем уровне" на хостинге.

2.1. Подготовка файлов
- Убедитесь, что у вас есть все файлы проекта:
  - index.php (главная страница)
  - about.php (страница "О нас")
  - contacts.php (страница "Контакты")
  - send_contact.php (обработка формы обратной связи)
  - login_user.php (страница входа)
  - register.php (страница регистрации)
  - user_profile.php (личный кабинет)
  - logout_user.php (выход из аккаунта)
  - admin.php (админ-панель)
  - add_to_cart.php (добавление в корзину)
  - checkout.php (оформление заказа)
  - images/ (папка с изображениями сухофруктов)
  - sql/dryfruits_shop_db.sql (файл дампа базы данных)

2.2. Загрузка файлов на хостинг
- Подключитесь к вашему хостингу через FTP (например, с помощью FileZilla):
  - Хост: (укажите адрес FTP, например, ftp.yourdomain.com)
  - Пользователь: (ваш FTP-логин)
  - Пароль: (ваш FTP-пароль)
  - Порт: 21 (обычно)
- Загрузите все файлы проекта в корневую папку (обычно это public_html или www).

2.3. Создание базы данных
- Зайдите в панель управления хостингом (например, cPanel).
- Найдите раздел "Базы данных MySQL" или "MySQL Databases".
- Создайте новую базу данных:
  - Название: dryfruits_shop_db
- Создайте нового пользователя для базы данных:
  - Пользователь: (например, dryfruits_user)
  - Пароль: (например, StrongPass123!)
- Привяжите пользователя к базе данных:
  - Выберите базу dryfruits_shop_db и пользователя dryfruits_user.
  - Дайте пользователю все права (ALL PRIVILEGES).
- Зайдите в phpMyAdmin (через панель управления хостингом).
- Импортируйте дамп базы данных:
  - Выберите базу dryfruits_shop_db.
  - Нажмите "Импорт" (Import).
  - Выберите файл sql/dryfruits_shop_db.sql из проекта.
  - Нажмите "Выполнить" (Go).

2.4. Настройка подключения к базе данных
- Откройте любой PHP-файл, где есть подключение к базе (например, index.php, admin.php).
- Найдите строку подключения:
  $conn = mysqli_connect("localhost", "root", "", "dryfruits_shop_db");
- Измените её, используя данные вашего хостинга:
  $conn = mysqli_connect("localhost", "dryfruits_user", "StrongPass123!", "dryfruits_shop_db");
  - Хост: обычно "localhost", но на хостинге может быть другой (например, mysql.yourdomain.com).
  - Пользователь: имя пользователя базы данных (dryfruits_user).
  - Пароль: пароль пользователя базы данных (StrongPass123!).
  - База: dryfruits_shop_db.
